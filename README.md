El presente proyecto consistió en el desarrollo de un sistema orientado a la gestión de servicios turísticos y ventas utilizando Python, con interacción mediante una interfaz de línea de comandos (CLI). La problemática identificada fue la dificultad para administrar de manera ordenada y eficiente la información de servicios, precios y ventas cuando estos procesos se realizan manualmente, lo que puede generar errores en los cálculos, pérdida de información y desorganización de los registros.

La solución implementada fue una aplicación desarrollada en Python que emplea listas como estructura de datos principal, permitiendo ingresar servicios, registrar ventas con numeración correlativa, validar cantidades, calcular precios de venta con IGV y determinar el total de cada transacción. Asimismo, el sistema incorpora operaciones de búsqueda y ordenamiento mediante algoritmos como búsqueda secuencial, búsqueda binaria, Quicksort y BubbleSort. Finalmente, la información es almacenada en archivos de texto (.txt) y todas las funcionalidades del sistema se presentan al usuario a través de un menú CLI.
